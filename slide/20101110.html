こんなことを思いついた

Node.js と chrome extension 使えば どのページでも動くリアルタイム Web アプリが作れるぞ

命名

Everywhere! (twitter のアプリケーションに登録できなかったので ! がついた)

とりあえずチャット

デモ

某 node.js 飲み会にて

Facebook の翻訳プロジェクトを 全てのページでできる アプリケーションを つくりたい。 via @jedschmidt

ぱくってつくってみた

閲覧中のページに、 インラインエディタを仕込んで 要素を編集できるようにした。 編集内容は他の人に送信。

その後・・

・共同作業するユーザの指定 ・redo / undo ・翻訳内容をサーバに保存 ・他の人の翻訳を選べるようにした ・Google 翻訳 / Bing 翻訳

アーキテクチャ

server: node.js(http/ws server) + redis(kvs) client: chrome extension
