Chat & Edit Everywhere!

Everywhere! enables you to chat on any webpage and edit any webpage's element!

About

Masahiro Hayashi (林正浩) Twitter: @masahiroh Node.js JP

What's Everywhere!?

すべての Web ページに Real-time Web app を 追加してしまおう という試み。

Client

Chrome Extension です。 これをインストールしてください。 http://bit.ly/everywhere-client どこにもアナウンスしない間に、星１つけられたし。。

ひとつめ

とにもかくにも..

まずは、チャットだろう
Extension をインストールした人は ページ右下の数字を クリックしてみてくださいね

が、、、

ページに Extension をインストールしてる人がいないと役に立たない。 共同作業時のコミュニケーション用途でつかったほうがよさげ。

というわけで、

エディタをつけてみよう。
