Chat everywhere!

everywhere! enables you to tweet and chat on any webpage.

How to

  1. everywhere! uses twitter account. If you don't have one, let's sign up.
  2. Download and install a client (Chrome extension).
  3. Click the everywhere!'s icon, and allow the client to access your twitter account.
  4. That's all! Let's chat on any webpage!
