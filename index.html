Translate and share any web page!

  • Everywhere! enables you to translate directly the page you visit and share it.
  • You can copy and modify others' translation including Google translation and Bing translation.
  • You can collaborate with others in real time.
  • You can also chat with people on the page.

Howto

Prepearetion

  1. You need an Twitter account.
  2. Install the Chrome extension, and allow it to access your account.

Read Translation

  1. Open any page (except for a page starts 'https://').
  2. Click the Everywhere! icon and display the toolbar then select translations.
  3. You can use Google Translate and Bing Translateg by default. If other people publish their translations, the author's name will be added to the dropdown menu and you can read it.

Translate

  1. Open any page that you want to translate (except for a page starts 'https://').
  2. Click 'Trnaslate by yourself'. If you select other's translation before you click this button, you can copy it and make your own.
  3. Click an element that its background color change when you mouse over it, you can edit the page browsing. The change will be applied when you unfocus it.
  4. At the same time, the data will be sent to the server and be published.

Collaborate

  1. If you want to collaborate with others, every collaborators need to open the same page.
  2. Click 'Collaborate' button, then online people's avators will be displayed. Select people you want to work with, and click 'OK'. The invitation dialog will be displayed on their browser. If they approve your invitation, your changing will be applied to each other.
  3. The sharing setting will be saved on the server, so you don't need to do this next time.

Chat

  1. You can chat with others with online people.
  2. If you click the number icon on the right bottom corner on the page, chat widget will be displayed. The number incicates the number of online people.
  3. If you check 'tweet', your message will be also tweeted with a short url of the page. Maybe you should ask your followers "Hey! Please translate this page!"
