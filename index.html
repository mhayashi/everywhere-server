Chat Everywhere! Everywhere! enables you to chat on any webpage!

Howto

  1. Install the Chrome Extension.
  2. Allow Everywhere! to access your Twitter account.
  3. Go to any webpage.
  4. Click the number icon at bottom right corner on the page.
  5. Send a message.
  6. The message will be send to the people on the same page and tweeted automatically with a short URL.
