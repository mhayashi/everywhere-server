好きなページをその場で翻訳してシェアしましょう。

  • Everywhere! は、開いているページを直接編集して翻訳を作成し、シェアできるサービスです。
  • 他の人の訳したもの（Google 翻訳や Bing 翻訳も使えます）をコピーして、自分の翻訳を作ることもできます。
  • リアルタイムに共同で翻訳作業をすることもできます。
  • ページ上でチャットもできます。

Howto

準備

  1. Everywhere! を使うには Twitter アカウントが必要です。
  2. Chrome extensionをインストールし、 Twitter アカウントへのアクセスを承認してください。

翻訳を読む

  1. 好きなページを開きます (https:// で始まるサイトは対象外です) 。
  2. Everywhere! のアイコンをクリックして、ツールバーを表示させます。「翻訳を選択」から翻訳を選択します。
  3. Google と Bing の自動翻訳が予め使えるようになっています。他の人が翻訳を公開している場合、作成者の名前がメニューに追加され、翻訳を選択できるようになります。

翻訳する

  1. 翻訳したいページを開きます (https:// で始まるサイトは対象外です) 。
  2. 「自分で翻訳」を押します。このボタンを押す前に他の翻訳を開いていた場合は、それを下地にして作業を始めることができます。
  3. 背景の色がかわるところでクリックすると、現在閲覧中のページを編集することができます。フォーカスを外すとページに反映されます。
  4. 反映と同時にサーバにデータが送信され、公開完了です。

共同で作業する

  1. 他の人と共同作業するには、まず、共同作業したい人と同時にそのページを開いている必要があります。
  2. 「共同作業」を押すと、現在オンライン中の人が表示されます。共有する相手を選択して「OK」を押すと、相手側に共同作業のダイアログが表示されます。「OK」を押すとお互いの編集内容が即時に反映されるようになります。
  3. 共有設定はサーバに保存されているので、次回以降はこの作業は必要ありません。

会話する

  1. 共同作業中はチャットで打ち合わせもできます。
  2. ページの右下の数字マークを押すと、チャット画面が表示されます。数字はオンライン中の人の数を示しています。
  3. 「Tweet」をチェックすると、そのページの短縮 URL つきで発言がツイートされます。「このページ翻訳して」ってお願いしてみてもいいかもしれませんね。
